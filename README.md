# zillow-python
A python wrapper for Zillow

```
import zillow
Z = zillow.Zillow()
```

## GetSearchResults
`Z.GetSearchResults("123 ABC Street","Bellingham, WA")`

## GetZestimate
`Z.GetZestimate("12345678")`

## GetChart
[Params](https://www.zillow.com/howto/api/GetChart.htm)
`Z.GetChart("48749425","dollar",600,300,"10years")`
